# AGENTS.md - IRandonation GitHub Profile Repository

> This file provides guidance for AI coding agents working in this repository.

## Repository Overview

This is a **GitHub Profile Repository** - a special repository named `IRandonation/IRandonation` that displays on the user's GitHub profile page. It is NOT a code project.

### Repository Structure

```
D:\Project\IRandonation/
├── .git/                    # Git repository data
├── Image/                   # Profile images and assets
│   ├── 1.gif               # Animated profile avatar
│   └── 56382608.png        # Static profile image
├── README.md               # GitHub profile page content
└── AGENTS.md               # This file - agent instructions
```

## Project Type

- **Type**: GitHub Profile Repository
- **Purpose**: Display personalized GitHub profile with stats, badges, and links
- **Tech Stack**: Markdown, HTML/CSS (embedded in README.md)
- **No Build/Lint/Test**: This is a documentation-only repository

## External Services Used

The README.md uses the following external services to generate dynamic content:

| Service | URL | Purpose | Status |
|---------|-----|---------|--------|
| Shields.io | `img.shields.io` | Badges and stats | ✅ Stable |
| Typing SVG | `readme-typing-svg.demolab.com` | Animated typing text | ⚠️ May timeout in China |
| Profile Views | `komarev.com/ghpvc` | Visitor counter | ✅ Stable |
| GitHub Raw | `raw.githubusercontent.com` | Profile images | ✅ Stable |

**Note**: Vercel/Heroku hosted services (github-readme-stats, trophy, streak-stats) have been replaced with shields.io for better accessibility in China.

## Code Style Guidelines

Since this repository only contains Markdown and embedded HTML/CSS:

### Markdown Style
- Use `<div align="center">` for centered content
- Use HTML comments `<!-- -->` for section labels
- Use inline styles for visual customization
- Keep sections separated with horizontal rules or styled dividers

### Image Guidelines
- Store profile images in `Image/` directory
- Reference via `https://raw.githubusercontent.com/IRandonation/IRandonation/master/Image/`
- Use HTTPS for all external image URLs

### Badge Style
- Use `style=for-the-badge` for main badges
- Use `style=flat-square` for technology badges
- Match color themes (currently using "radical" and "onedark" themes)

## Making Changes

### To Update Profile Stats
- No action needed - stats are dynamically generated by external services
- Stats update automatically based on GitHub activity

### To Add/Modify Sections
1. Edit `README.md`
2. Follow existing HTML structure with inline styles
3. Use existing service APIs for new stat cards
4. Commit changes directly (no build step required)

### To Update Images
1. Add new images to `Image/` directory
2. Update README.md references if needed
3. Commit and push - changes are immediate

## Important Notes

- **No Testing Required**: This is a static profile page
- **No Linting Required**: Markdown and embedded HTML don't require linting
- **Direct Commits**: Changes can be committed directly to `master` branch
- **Immediate Updates**: GitHub profile updates within minutes of pushing
